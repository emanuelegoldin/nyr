# New Year Resolution Bingo

An application for users to create a bingo card given N new year resolutions and compete with other users.

## User Stories

- As a user, I want to register creating a profile with a username/email and a password
- As a user, I want to login to then see my profile
- As a user, when I create a profile, I need to confirm my identity throguh email
- As a user, I want to be the only one capable of modifying my personal data on the website
- As a user, I want to decide which information to make publicly available on the website
- As a user, I want to add, update, and delete resolutions
- As a user, I want to create a bingo card from the resolutions I defined
- As a user, I want to update the bingo card: when I complete a resolution I want to sign it as "completed"; if I mistakenly signed a reslution as completed, I want to correct my error and set it as "to complete"
- As a user, I want to see the current state of my bingo card. I want to see also other user's bingo cards for the users competing with me
- As a user, I want to join a group. In the group I can define one resolution for each participant and a group challenge that affect the whole group
- As a user when I update my bingo card, I want the other users in the group to see my update
